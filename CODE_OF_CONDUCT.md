# Código de Conduta

## Nosso compromisso

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/), versão 2.1, como código de conduta. Participantes — mantenedores e contribuidores — comprometem-se a tornar a participação no projeto uma experiência livre de assédio para todas as pessoas, independentemente de qualquer característica pessoal.

## Resumo prático

- Seja respeitoso em issues, PRs e discussões;
- Crítica técnica é bem-vinda (inclusive dura) — ataque ideias, nunca pessoas;
- Discussões jurídicas e metodológicas devem permanecer no escopo do benchmark;
- Não inclua dados pessoais ou confidenciais em issues públicas.

## Aplicação

Comportamentos abusivos podem ser reportados em particular para **thalesandradees@gmail.com**. Todas as denúncias serão revisadas e tratadas com confidencialidade.
